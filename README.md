# CRUD em Python com SQLite

Projeto de estudo em Python que implementa um CRUD completo usando SQLite.

## Funcionalidades
- Criar usuário
- Listar usuários
- Buscar por ID
- Atualizar usuário
- Excluir usuário
- Validação de email via API (Hunter)

## Tecnologias
- Python
- SQLite
- Requests

## Como executar
```bash
python main.py
