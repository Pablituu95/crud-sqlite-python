# CRUD em Python com SQLite

Projeto de estudo em Python que implementa um CRUD completo usando SQLite.

## Funcionalidades
- Criar usuário
- Listar usuários
- Buscar por ID
- Atualizar usuário
- Excluir usuário
- Validação de email via API (Hunter)

## Tecnologias
- Python
- SQLite
- Requests

## Como executar
```bash
python main.py

## Para conseguir a api_key
Acesse o site: https://hunter.io/dashboard, crie um usuário e vá para https://hunter.io/api-keys, e então gere uma chave api.
